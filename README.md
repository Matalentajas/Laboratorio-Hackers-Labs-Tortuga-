# Laboratorio-Hackers-Labs-Tortuga-
Este informe documenta el proceso de Penetration Testing (Pentesting) realizado sobre la máquina virtual objetivo "Tortuga". El ejercicio se centró en la metodología de descubrimiento de hosts, ingeniería de wordlists y la explotación de vulnerabilidades de autenticación y Escalada de Privilegios Local (LPE).
